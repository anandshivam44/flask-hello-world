# This is a Flask Hello World Application

### Prerequisites
- python3
- Flask
  
### To Install Flask
```
sudo pip3 install Flask
```
### Start Applictaion
```
python3 hello.py
```
Visit [127.0.0.1:5000](http://127.0.0.1:5000/) or [localhost:5000](http://localhost:5000/) in Browser or Postman
